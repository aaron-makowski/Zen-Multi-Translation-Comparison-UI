# Contributing

Thanks for your interest in contributing! To get started:

1. Fork the repository and create a new branch for your changes.
2. Install dependencies with `pnpm install`.
3. Make your changes and add tests when applicable.
4. Run `pnpm lint` and `pnpm test` to ensure everything passes.
5. Submit a pull request describing your changes.

We welcome bug reports, feature requests, and pull requests. If you have any questions, feel free to open an issue.
